# daily-ai-prompt — Setup

> One useful AI prompt per day, optionally generated by your own model.

## Install (one click)

1. Install [Übersicht](https://tracesof.net/uebersicht/) and run it once.
2. Double-click `install.command` (or run `./install.sh` in Terminal).
   It copies `daily-ai-prompt.widget` into your Übersicht widgets folder, installs any
   helpers, and walks you through any configuration.

The installer is safe to re-run; it just refreshes the install in place.
To install by hand instead, unzip `daily-ai-prompt.widget.zip` into
`~/Library/Application Support/Übersicht/widgets/`.

## Configuration

The widget works out of the box using a built-in library of high-quality
prompts. To have a model write a fresh, personalized prompt each day, the
installer can set up an API key:

- **Claude** — `~/.config/widgetsuite/anthropic.key`
- **ChatGPT** — `~/.config/widgetsuite/openai.key`
- **Gemini** — `~/.config/widgetsuite/gemini.key`

`install.sh` prompts for one of these and stores it (mode `600`). The active
provider is written to `~/.config/widgetsuite/ai-provider.txt`.

**Switch models any time** by clicking the logo in the widget's corner: the
current model's mark is shown, and clicking it reveals the others. Your choice
is saved and the next prompt is generated by that model.

Add a short bio at `~/.config/widgetsuite/profile.txt` (the installer can do
this) to make generated prompts more personal.

## Fonts

For the intended look, install **Instrument Serif**, **Geist**, and
**Geist Mono**. System fonts are used as a fallback otherwise.
